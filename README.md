## 🚀 Proyecto en el que contribuí

### Sistema de administración de órdenes de mantenimiento de equipos

Participé en el desarrollo de un sistema web para gestionar las órdenes de mantenimiento de equipos de cómputo del área técnica.

**Tecnologías utilizadas**

* Laravel
* Vue.js
* PHP

**Funciones principales**

* Registro de equipos recibidos en mantenimiento
* Creación y gestión de órdenes de servicio
* Seguimiento del estado de reparación
* Historial de mantenimiento de equipos

🔗 Repositorio del proyecto
https://github.com/SebastiiianG/mantenimiento_ceca
